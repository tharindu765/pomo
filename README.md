# Pomo - Personal Portfolio Website

A modern, dark-themed personal portfolio website built for **Tharindu Randika**, an AI Engineer Intern and Full-Stack Developer.

## Features

- **Dark Theme Design**: Inspired by the Matias theme with neon yellow (#CDFF00) accents.
- **Responsive Layout**: Fully responsive design that works on mobile, tablet, and desktop.
- **Custom Assets**: 
  - 3D illustrated character for the About section.
  - Custom project thumbnails.
  - Vector SVG logo.
- **Interactive Elements**:
  - Smooth scrolling navigation.
  - Hover effects on cards and buttons.
  - Mobile hamburger menu.
  - Scrolling skills ticker.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Flexbox, CSS Grid, and animations.
- **JavaScript**: DOM manipulation for interactions and scroll effects.
- **Google Fonts**: Inter can Outfit typefaces.
- **Font Awesome**: Icons.

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/tharindu765/pomo.git
   ```
2. Open `index.html` in your browser.
